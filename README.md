# 项目介绍

此项目类似淘宝那样的一个商城购物平台的小项目，主要是从数据库获取数据展示在页面。用户可以登陆、注册。用户可以将商品加入到自己的购物车，进行购买商品，然后生成订单。目前的前端项目还在完善中，个人感觉还不是很完善。佛系添加吧
